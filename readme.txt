== Description ==
Quick View by Code Astrology for WooCommerce is a lightweight WooCommerce based plugin. A user can easily view a product without redirection. The advanced slider is used for navigating gallery products. Lots of options and live customizable.

== Settings==
Go to Appearance > Customize > Quick View Settings



== Features ==

* Live Customize style.
* Work on all types of products.
* Cool animations for Quick View modal.
* Almost everything is customizable.
